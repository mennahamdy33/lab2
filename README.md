## Tell me how to remove dev branch locally and remotely.
### Locally:
git branch -d dev
### remotely:
git push origin :dev
## Tell me how to list tags.
git tag --list
## Tell me how to delete tag locally and remotely
### To delete local tags
git tag -d v1.7
### To delete remote tag
git push origin --delete v1.7
